## Installation

```
$ git clone https://github.com/sefatascan/examples-accounting.git
$ cd accounting
```

### docker
```
$ docker-compose up -d
```

### maven
```
$ ./mvnv clean install
$ java -jar target/accounting-0.0.1-SNAPSHOT.jar
```
